# Prework Study Guide Webpage

## Description


This Webpage was build to understand how HTML,CSS and Javascript work together while providing a a prework study guide website for students 

The webpage organises the prework study elements into a single page and helps students to know what will be the topics they will learn in the prework studies.

## Usage

Each section contains notes about HTML,CSS,Javascript,Git.

If we open the DevTools console, we can see the JavaScript running. It will list the topics we learned from prework and choose one topic for us to study first.
